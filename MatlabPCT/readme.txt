# MatlabPCT Implementation
This is the MatlabPCT implementation of the Neural Network
### Compilation
This is written for MatlabPCT.

It can be run with the following command: </br>
**ex4**
### Running the code on the discovery cluster
In this repo there is a script called bsubmit.sh. The user only needs to **modify the script for their name and directory** on the discovery cluster and execute:</br>
**bsub < bsubmit.sh**
